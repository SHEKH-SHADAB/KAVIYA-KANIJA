# KAVIYA-KANIJA

KAVIYA-KANIJA is an Android application built using Kotlin and Jetpack Compose. The app is designed to provide a clean and modern reading experience for Kannada kavite (poems) with an attractive UI and category-based navigation.

---

## Features

* Modern Android UI using Jetpack Compose
* Clean and responsive design
* Category-based poem browsing
* Search functionality
* Reader screen for viewing kavite content
* MVVM architecture
* Organized UI and data layers
* Material Design components

---

## Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio
* MVVM Architecture
* Gradle (KTS)

## Installation

1. Clone the repository

```bash
git clone https://github.com/SHEKH-SHADAB/KAVIYA-KANIJA.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the application on an emulator or physical device
